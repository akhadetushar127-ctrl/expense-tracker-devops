# 🚀 Expense Tracker DevOps Project

A full-stack Expense Tracker REST API built using Flask, Docker, Nginx, and monitoring tools.

---

## 📌 Features
- Add / View / Delete expenses
- REST API using Flask
- Docker containerization
- Docker Compose setup
- Nginx reverse proxy
- Prometheus monitoring
- Pytest test cases

---

## 🛠 Tech Stack
- Python (Flask)
- Docker & Docker Compose
- Nginx
- Prometheus
- PostgreSQL (optional)
- Redis (optional)

---

## 🚀 Run Locally

```bash
docker compose up --build
